Плата FP_01v3. 
Cделана в OrCAD 16.6.

Предназначена для использования совместо с АВАНТ всех модификаций работающих
от 220В.

Изменения:
Установлены самовосстанавливающиеся предохранители.
Добавлены демпфирующие цепочки параллельно симфазному дросселю.
Заложена возможность установки варисторов после каждого каскада.

Недостатки:
Варисторы установлены на землю. Которые к тому же периодически выгорают (-ли ?!)
на подстанции.
